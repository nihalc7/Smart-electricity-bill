About the Project

This project focuses on monitoring electricity consumption in real time and estimating the electricity bill. It helps users understand their power usage and reduce unnecessary energy consumption.

Objectives

  • To measure real-time electricity consumption
  • To estimate electricity bill accurately
  • To create awareness about energy saving
  • To provide a user-friendly monitoring system

Methodology

The system uses current and voltage sensors to measure power consumption. The data is processed using a microcontroller and displayed through a web interface.

Hardware & Software Used

  • Microcontroller (Arduino / ESP32)
  • Current Sensor
  • Voltage Sensor
  • Wi-Fi Module
  • HTML, CSS, JavaScript

Frequently Asked Questions (FAQ)

Q1. Is this bill calculation accurate like KSEB?
Yes. The system follows KSEB domestic slab tariff and estimates the 2‑month bill similar to the actual meter reading.

Q2. Why is real-time data useful?
KSEB provides bills every two months, but this system shows usage and cost instantly.

Q3. Can tariff rates be changed in the future?
Yes. Tariff values are configurable and can be updated based on government policies.

Q4. Is AI used in this system?
AI-based prediction analyzes usage patterns to forecast weekly and bi‑monthly bills.
